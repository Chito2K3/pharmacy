Setup & Configuration

Connect to Google Sheets backend and configure the system

1

Create a Google Sheet

Go to sheets.google.com → create a new sheet. Rename tabs to: Items, Reorders, Users. Then import your extracted items.json and reorders.json files using the import tool or a script.

2

Deploy Google Apps Script API

In your Google Sheet → Extensions → Apps Script. Paste the backend code below, click Deploy → New Deployment → Web App. Set access to "Anyone".

// Google Apps Script backend (paste this in Apps Script editor) function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet(); const action = e.parameter.action; if (action === 'getItems') return getItems(sheet); if (action === 'getReorders') return getReorders(sheet); if (action === 'getUser') return getUser(sheet, e.parameter.email); return ContentService.createTextOutput(JSON.stringify({error:'Unknown action'})) .setMimeType(ContentService.MimeType.JSON); } function getItems(ss) { const data = ss.getSheetByName('Items').getDataRange().getValues(); const headers = data[0]; const rows = data.slice(1).map(row => { const obj = {}; headers.forEach((h,i) => obj[h] = row[i]); return obj; }); return ContentService.createTextOutput(JSON.stringify(rows)) .setMimeType(ContentService.MimeType.JSON); } // Add getReorders, getUser functions similarly...
3

Paste Your Apps Script URL

After deployment, copy the Web App URL and paste it here. The dashboard will fetch live data from your Google Sheet.

4

Deploy This Dashboard Free on Netlify

Save this HTML file → go to netlify.com → drag the file into the deploy zone. Your dashboard will be live at a free *.netlify.app URL in seconds. No server, no cost, no credit card needed.

5

Manage Users

Add user emails and roles to the Users sheet in Google Sheets. The system checks this on login. Roles: Admin, Dispensing, Storage.
